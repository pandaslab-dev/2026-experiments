# bear game

a small 2d platformer where you play as a bear, collect fish, and avoid slimes.

## why i built it

this started as me following the brackeys godot tutorial, but i wanted to make it feel more like my own thing. i kept the core learning structure from the tutorial and added a bear theme, different art, and my own little vibe while figuring out how platformers come together in godot.

## tech stack

- godot 4.2
- gdscript
- pixel art, sound effects, and a web export

## current features

- run and jump platforming controls
- animated bear player art and bear-themed presentation
- fish pickups with score tracking
- slime enemies and a quick respawn on death

## what i learned

- how player movement, gravity, and animation states work in godot
- how to connect pickups, ui text, and simple game state
- how to follow a tutorial without copying it exactly and still make the project feel personal

## what i would improve next

- clean up some naming and ui polish
- add more level variety and enemy moments
- make the movement feel even tighter
- add a simple start screen and finish screen
