paint-ball

simple as possible html/css/js experiment that pulls one real score from the ESPN api every time you refresh!

practice for fetch/json and responsive ui

if no live game shows up, it falls back to the most recent final

if nothing usable comes back from the feeds, it shows simple x placeholders instead of fake data

it is hosted at paint-ball.pandaslab.dev
